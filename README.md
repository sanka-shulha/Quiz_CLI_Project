\# Консольна вікторина (CLI) — PostgreSQL



\## Опис проєкту

Консольний застосунок для проходження вікторин з використанням бази даних PostgreSQL.

Користувач може зареєструватися, увійти в систему, проходити вікторини за категоріями, переглядати свої результати та топ-20, а також експортувати свої результати у CSV файл.

Дані зберігаються в базі даних. Файли використовуються тільки для експорту звітів.



-------------------------



\## Використані технології



\- Python 3

\- PostgreSQL 18

\- pgAdmin 4

\- psycopg2 (підключення до БД)



------------------------



\## Структура бази даних

\### Таблиці:



\### 

\- users

\- id

\- login (унікальний)

\- password

\- birth\_date

\- is\_admin (BOOLEAN)



\### categories

\- id

\- name (унікальна)

\- is\_active (BOOLEAN)



\### 

\- questions

\- id

\- category\_id (FK)

\- text

\- option\_a

\- option\_b

\- option\_c

\- option\_d

\- correct\_option



\### 

\- results

\- id

\- user\_id (FK)

\- category\_id (FK)

\- score

\- total\_questions

\- created\_at



-------------------------------



\## Реалізована функціональність



\### Основні можливості



\- Реєстрація користувача

\- Вхід у систему

\- Старт вікторини за категорією

\- Підрахунок правильних відповідей

\- Збереження результату в БД

\- Перегляд власних результатів

\- Топ-20 результатів

\- Зміна пароля

\- Зміна дати народження

\- Експорт моїх результатів у CSV



-----------------------------



\## Бонусні можливості



\- Змішана вікторина (20 випадкових питань з різних категорій)

\- Адміністратор:

&nbsp; - поле is\_admin

&nbsp; - окрема адмін-панель

&nbsp; - активація / деактивація категорій (is\_active)



Неактивні категорії не відображаються для звичайних користувачів.



-----------------------------



\## Ініціалізація бази даних



Створити базу даних у pgAdmin з назвою quiz\_shulha

Виконати файл schema.sql

Виконати файл seed.sql

Налаштування підключення до бази даних (config.py)



DB\_NAME = quiz\_shulha

DB\_USER = postgres

DB\_PASSWORD = PASSWORD

DB\_HOST = localhost

DB\_PORT = 5432



-----------------------------



\## Запуск програми



Встановити бібліотеку psycopg2-binary

Запустити файл main.py



Демонстраційний сценарій

\- Зареєструвати користувача

\- Увійти

\- Пройти вікторину

\- Переглянути свої результати

\- Подивитися топ-20

\- Експортувати результати у CSV



У разі входу як адміністратор — вимкнути категорію та показати, що вона не відображається
